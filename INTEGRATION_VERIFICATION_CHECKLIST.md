## 1. API 클라이언트 통합 검증

### 1.1 기본 API 클라이언트 설정

- [ ] 중복 경로 문제가 해결됨 (`/api/api/...` 형태의 문제 없음)
- [ ] 토큰 관리 기능이 올바르게 구현됨 (저장, 갱신, 삭제)

### 1.3 사용자 관련 API 연동

- [ ] 사용자 잔액 조회 API가 올바르게 연동됨 (`/api/users/balance`)
